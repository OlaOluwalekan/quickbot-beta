function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet("World")); // Output: "Hello, World!"

Explanation:

  • Function declaration: function greet(name) { ... } defines a function named greet that takes a single argument called name.
  • String interpolation: return Hello, ${name}!; creates a string that includes the value of the name argument.
  • Function call: console.log(greet("World")); calls the greet function with the argument "World" and logs the returned string to the console.

How to Use:

  1. Create a new Next.js project (or add this code to an existing one).
  2. Create a new component file (e.g., MyComponent.tsx) and paste the code.
  3. Import the component into your main app file (e.g., pages/index.tsx) and render it.
  4. Run your Next.js project.
